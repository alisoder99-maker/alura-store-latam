# Análisis de rendimiento de tiendas - Alura Store Latam

## Descripción del proyecto

Este proyecto fue desarrollado como parte del desafío de **Alura Latam**, con el objetivo de analizar el desempeño de cuatro tiendas de la cadena **Alura Store** y recomendar cuál de ellas debería vender el Sr. Juan para iniciar un nuevo emprendimiento.

Para ello, se realizó un análisis de datos utilizando Python, Pandas y Matplotlib, considerando indicadores clave como ingresos, categorías más vendidas, calificaciones de clientes, productos más y menos vendidos y costo promedio de envío.

---

## Objetivo

Evaluar el rendimiento de las cuatro tiendas a partir de diferentes métricas para identificar cuál presenta el desempeño menos eficiente y, con base en ello, emitir una recomendación final sustentada en datos.

---

## Análisis realizados

Durante el desarrollo del proyecto se realizaron los siguientes análisis:

- **Ingreso total por tienda**
- **Ventas por categoría**
- **Calificación promedio de clientes por tienda**
- **Productos más vendidos y menos vendidos**
- **Costo de envío promedio por tienda**
- **Visualización de datos mediante gráficos**
- **Análisis geográfico de ventas** *(opcional, si fue desarrollado)*
