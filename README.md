# alura-store-latam
# Análisis de tiendas - Alura Store Latam

## Descripción
Este proyecto analiza el desempeño de cuatro tiendas de Alura Store con el objetivo de recomendar cuál debería vender el Sr. Juan para iniciar un nuevo emprendimiento.

## Objetivo
Evaluar las tiendas a partir de:
- Ingreso total
- Ventas por categoría
- Calificación promedio
- Productos más y menos vendidos
- Costo de envío promedio

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Estructura del proyecto
- data/: archivos CSV
- notebooks/: análisis del proyecto
- images/: gráficos generados
- README.md

## Cómo ejecutar
1. Abrir el notebook en Google Colab
2. Cargar los archivos CSV
3. Ejecutar las celdas en orden
4. Revisar los gráficos y el informe final
